🐍 方案一：Zeabur Python Cron 最推薦，一次設定永久自動
✅ 完整流程：Zeabur 每 2 天 13:30 自動觸發 → Claude 生成文案 → Gemini 生成配圖 → 自動上傳到 FB 粉絲專頁
  1. 點下方「⬇️ 下載 Python 程式」
  2. 前往你的 Zeabur Dashboard → 新增 Project → 新增 Python Service
  3. 上傳 main.py 與 requirements.txt
  4. 在 Service 的 Environment Variables 填入：
    CLAUDE_API_KEY / FB_PAGE_TOKEN / FB_PAGE_ID
  5. Zeabur 自動部署，每 2 天 13:30 自動發文 ✅
🔄 方案二：n8n 自架於 Zeabur 視覺化流程，進階彈性
n8n 是開源自動化工具，部署在你的 Zeabur 上就完全免費，還能視覺化拖拉設定流程。
  1. Zeabur → Marketplace → 搜尋 n8n → 一鍵部署
  2. 在 n8n 建立 Workflow：Schedule → HTTP Request (Claude API) → HTTP Request (Gemini) → HTTP Request (FB Graph API)
  3. 設定排程每 2 天 13:30 執行
  4. 點下方「⬇️ 下載 n8n Workflow JSON」匯入即可
🔑 Facebook Token 取得步驟
  1. 前往 developers.facebook.com/apps → 建立應用程式 → 選「其他」→「商業」
  2. 左側選單 → Graph API Explorer
  3. 右上角選你的粉絲專頁（Page）
  4. 點「新增權限」→ 勾選 pages_manage_posts + pages_read_engagement
  5. 點「生成存取權杖」→ 複製 Token
  6. ⚠️ 短期 Token 只有 1 小時，需換成長期 Token（我幫你生成換 Token 的請求）
